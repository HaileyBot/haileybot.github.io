This is HaileyBot

Global chat, advertising, and images!

Global Chat

The global chat feature of this bot can link a channel from your Discord server to a multiserver chat - any message sent from the connected channel on your server will send to all the other connected servers, and vice versa. To use the global chat, we do require you to accept the EULA on our main Discord server. This is to avoid problems that can potentially get this bot shut down. By breaking the terms of the EULA, you are breaking the law, and legal action can be taken against you. To set up a global chat link on your server, you need to send a request to @cheesits456#6494 via direct message on Discord.

Advertising

The advertising feature of this bot allows each user to create an advertisement for their server. After adding the bot to your server, you only have to do one simple command to set up an advertisement. Server ads can only be created by server admins, and must follow the rules.

Images

The image feature of this bot lets you send fun images from the bot. Images are limited to 200 pixels in height so they don't take up too much space and aren't too spammy.

Rules

General rules for using the global chat - failure to comply will result in messages from your account being blocked.

[1] No NSFW content - this includes porn/nudity and gore, text and images.

[2] No excessive spam

[3] No unrelated botspam - bot commands are fine but if they're interrupting a conversation needlessly then stop. If there's no convos currently happening then use whatever bot commands you want, as long as they comply with the other rules.

[4] No hate speech - if you're a racist, homophobic, bigoted, or extremist, go somewhere else

[5] No disrespect - if you're bothering someone, be respectful and stop what you're doing. Just be respectful in general :)

[6] This is an English chat - keep it English

[7] No bypassing the chat filter - doing so will result in an immediate ban, no warnings
